This website contains a collection of cafes, or places where you can get drinks, near Hunter College. This includes well-known chain cafes, and some local cafes.

It's also honestly a good way for me to remember the places that aren't that popular.

I really like getting iced drinks. Is it for the aesthetic and because it's trendy? Maybe, I don't doubt that. Is it a fixation? Because I need to have something in my hands at all times?

I will say I've never had amazing tasting drinks in NYC. The coffee has never wowed me, but Birch Coffee was pretty close to the best coffee I've had.

Best coffee I've had was up in Beacon actually, so sorry NYC.
